Tabbed Content

This page is for: Content Managers Designers Developers

Tabs are used to group similar pages into a single page when there is a need to explain content heavy topics. If a generic content page requires further pages or information, Tabs are used in place of a traditional left or right side submenu. Tabs, in essence, allow ‘pages’ to be embedded within a ‘page’.

Unlike accordions (which are restricted to text only) tabs are able to contain any content widgets, with the exclusion of accordions.

No content should be added to the page after a Tab panel (Tabs should be the last element on the page) and only one set of Tabs can be added to a page.

Tabs are listed horizontally, so only a certain number of tabs will fit within the width of the content area. Therefore, the naming of Tabs need to be kept to as short as possible.

Tabs revert to accordions when in mobile mode.

Example:


  • Home
  • Profile
  • Messages
  • Settings
